Cómo crear mis credenciales de Desarrollo

Para poder acceder a nuestra API es necesario poseer un API Key y un Access Token. Para obtenerlos se deben seguir los siguientes pasos

  1. Ingresar a Portal de Desarrolladores
  2. Crear una nueva aplicación o utilizar una existente
  3. Ingresar a la aplicación creada
  4. Obtener el acceso a una Entidad mediante la opción "Solicitar Acceso" o utilizar un acceso ya concedido mostrado en el listado de "Entidades Autorizadas"

Habiendo obtenido ambos keys se puede proceder a generar las solicitudes a la API descriptas en esta documentación.

Cabeceras (Headers)

Con todas las solicitudes a nuestra API deberá enviar ambas keys las cuales permitiran el acceso a los recursos.

x-api-key: Clave API de su aplicación
x-access-token: Token de Acceso a la entidad para la cual desea realizar la operación.
