Mobbex²

El SDK para Embeber le permite embeber de manera fácil y rápida nuesta solución de checkout.

Creación

La inserción del botón Mobbex puede hacerse creando un checkout desde el lado Servidor o una Suscripción por API o via la Consola de Mobbex.

Implementación

Versión

La versión actual es: 1.0.8.

Insertar en el sitio de manera Fija en la Cabecera

<script src="https://res.mobbex.com/js/embed/mobbex.embed@1.0.8.js"></script>

Insertar en el sitio dinámicamente en el Body

Cargá nuestra librería Javascript en tu sitio:

<html>
<body>
  <script type="text/javascript">
    var script = document.createElement('script');
    script.src = `https://res.mobbex.com/js/embed/mobbex.embed@1.0.8.js?t=${Date.now()}`;
    script.async = true;
    script.addEventListener('load', () => {
      // Realizá la acción que sea necesaria aca :)
      // renderMobbexButton();
      // initMobbexPayment();
    });
    document.body.appendChild(script);
  </script>
</body>
</html>

Creación y Eventos

Al renderizar podrás utilizar un checkout generado previamente en el servidor. Ante un pago se ejecutarán 2 callbacks: onPayment y onClose. En onClose podrás detectar que el pago fue realizado con la variable "cancelled" Si "cancelled" es false es porque el pago fue realizado correctamente.

Renderizar el botón

  1. Crear un div con el ID mbbx-button
<div id="mbbx-button"></div>
  1. Crear las Opciones basado en el tipo de Pago: Checkout o Suscripciones

Opciones para Checkout

var options = {
  id: '{ID Checkout}',
  type: 'checkout',
    onResult: (data) => {
      // OnResult es llamado cuando se toca el Botón Cerrar
      window.MobbexEmbed.close();
    },
    onPayment: (data) => {
      console.info('Payment: ', data);
    },
    onOpen: () => {
       console.info('Pago iniciado.');
    },
    onClose: (cancelled) => {
       console.info(`${cancelled ? 'Cancelado' : 'Cerrado'}`);
    },
    onError: (error) => {
       console.error("ERROR: ", error);
    }
}

Opciones para Suscripciones

var options = {
  id: '{ID Suscripción}',
  sid: '{ID Suscriptor}',
  type: 'subscriber_source',
    onResult: (data) => {
      // OnResult es llamado cuando se toca el Botón Cerrar
      window.MobbexEmbed.close();
    },
    onPayment: (data) => {
      console.info('Payment: ', data);
    },
    onOpen: () => {
      console.info('Pago iniciado.');
    },
    onClose: (cancelled) => {
      console.info(`${cancelled ? 'Cancelado' : 'Cerrado'}`);
    },
    onError: (error) => {
      console.error("ERROR: ", error);
    }
}
  1. Inicializar el botón renderizando el mismo cuando la libreria está disponible.
function renderMobbexButton() {
  window.MobbexEmbed.render(options, '#mbbx-button');
}

Creación y Apertura por Código

  1. Crear un div con el ID mbbx-container
<div id="mbbx-container"></div>
  1. Ejecutar la creación del pago mediante el API provista
function renderMobbexButton() {
  var mbbxButton = window.MobbexEmbed.init(options);

  mbbxButton.open();
}

Opciones

Las siguientes opciones pueden ser provistas al botón Mobbex en la renderización o inicialización:

Inicializadores

type: Tipo de Pago Embebido: Sólo acepta 2 opciones: Checkout o Cambio de Medio de pago para Suscriptores.
id: ID de Checkout generado en el servidor o ID de Suscipción dependiendo el tipo. sid: ID de Suscriptor sólo para Suscripciones.

Callbacks

onPayment: (data) => { }
onError: (error) => { }
onOpen: () => { }
onClose: (cancelled) => { }

Personalización

button: Objecto Javascript con las siguientes opciones

{
    text: 'Pagar con Mobbex',
    image: 'https://res.mobbex.com/images/sources/mobbex.png',
    backgroundColor: 'rgb(137,0,255)',
    textColor: 'rgb(255, 255, 255)',
    textSize: '1rem'
}

Creación de Checkout para Embeber

Sólo se deben crear Checkouts del lado Servidor y compartir únicamente el ID o UID generado por Mobbex.

Checkout Ejemplo

Esto es un ejemplo de la utilización de las opciones especiales en PHP, para más ejemplos ver la documentación de Checkout.

<?php

$request = new HttpRequest();
$request->setUrl('https://api.mobbex.com/p/checkout');
$request->setMethod(HTTP_METH_POST);

$request->setHeaders(array(
  'cache-control' => 'no-cache',
  'Content-Type' => 'application/json',
  'x-lang' => 'es',
  'x-access-token' => 'd31f0721-2f85-44e7-bcc6-15e19d1a53cc',
  'x-api-key' => 'zJ8LFTBX6Ba8D611e9io13fDZAwj0QmKO1Hn1yIj'
));

$request->setBody('{
    "total": 100.2,
    "currency": "ARS",
    "reference": "2982-2XtPXlgSaWccqUyobuv4sEmLYMV0N6oX6MoridMw",
    "description": "Descripción de la Venta",
    "items": [{
        "image": "https://www.mobbex.com/wp-content/uploads/2019/03/web_logo.png",
        "quantity": 2,
        "description": "Mi Producto",
        "total": 50
    }, {
        "image": "https://www.mobbex.com/wp-content/uploads/2019/03/web_logo.png",
        "quantity": 1,
        "description": "Mi otro producto",
        "total": 50.2
    }],
    "options": {
    "domain": "midominio.com",
    "button": true,
    "embed": true
    },

try {

}
