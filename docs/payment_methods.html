Métodos de Pago y Cuotas

Listado de Métodos de Pago

Implementación Simple sin API-Keys

Esta API es Pública y no requiere del uso de API Key aunque debe generarse la conexión con la cuenta utilizando el botón de "Utilizar Widget / API" en la consola de Mobbex

GET: https://api.mobbex.com/p/sources/list/{codigo}?total={total}

Parametros:

Obligatorios:

  • codigo: Código de acceso generado por la consola de Mobbex

Implementación con API-Keys

URL: https://api.mobbex.com/p/sources

GET /p/sources HTTP/1.1
Host: https://api.mobbex.com
Content-Type: application/json; charset=utf-8
x-api-key: {API KEY}
x-access-token: {ACCESS TOKEN}

Parametros de Tipo Query

  • total: Monto de la operación.
Ejemplos de Total:
- 100
- 100.10
- 100.00

Ejemplo:

https://api.mobbex.com/p/sources/list/e463a332-242f-4f49-9962-714acb41f028?total=12500
  • Código: e463a332-242f-4f49-9962-714acb41f028
  • Total: 12500

Respuesta

  • result: Resultado de la operación
  • data: Array de datos de la operación.

Data

  • source: Posee los datos del medio de pago: nombre y referencia
  • view: Posee el tipo de Medio de Pago
  • installments: Nodo Opcional y sólo disponible en métodos de pago que poseen Cuotas. Array de Cuotas. Si el método posee cuotas encontrará un nodo "list"

Installments -> List

  • count: Cantidad real de Cuotas, es el divisor del monto total
  • name: Nombre del sistema de cuotas
  • description: Descripción opcional del sistema de cuotas.
  • totals: Nodo donde se indican los recargos, descuentos y cuotas
{
    "totals": {
        "currency": {
            "value": "ARS",
            "label": "Peso Argentino",
            "symbol": "$",
            "hidden": false
        },
        "installment": {
            "amount": 12500,
            "count": 1
        },
        "total": 12500,
        "financial": {
            "percentage": 0,
            "amount": 0
        }
    }
}

Imágenes de Métodos de Pago

Los métodos de pago posee un nodo llamado "reference" que identifica los métodos de pago en Mobbex. Para obtener las imágenes se debe utilizar este nodo.

URL:

https://res.mobbex.com/images/sources/{reference}.{format}

Tipo:
GET

Parametros:

Obligatorios:

  • reference: Referencia del método de pago obtenida desde la API del Listado
  • format: El formato de la imagen. Sólo admite png o jpg

Ejemplo

https://res.mobbex.com/images/sources/visa.frances.png
