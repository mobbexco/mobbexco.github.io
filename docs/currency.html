Conversión de Monedas

Conversión de Monedas

Implementación Simple sin API-Keys

Esta API requiere del uso de API Key y Access Token

Los valores devueltos por la API se actualizan cada 10 minutos.

GET: https://mobbex.com/p/currency/convert

Cabeceras

  • x-api-key: {API KEY}
  • x-access-token: {ACCESS TOKEN}

Parametros:

Obligatorios:

  • from: Moneda desde la cual se quiere convertir. Valores Admitidos: USD
  • to: Moneda a la cual se quiere convertir. Valores Admitidos: ARS
  • total: Valor a Convertir. Decimales separados por coma
Ejemplos de Total:
- 100
- 100.10
- 100.00

Ejemplo:

https://mobbex.com/p/currency/convert?from=USD&to=ARS&total=10
  • from: USD
  • to: ARS
  • total: 10

Respuesta

  • result: Resultado de la operación
  • data: Array de datos de la operación.
{
    "result": true,
    "data": {
        "from": "USD",
        "to": "ARS",
        "total": "10",
        "result": 466,
        "rate": 46.6
    }
}
