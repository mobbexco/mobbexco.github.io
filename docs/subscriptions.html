Suscripciones

URL

https://api.mobbex.com/p/subscriptions

Creación

Cabeceras (Headers)

  • x-api-key: Clave API de su aplicación
  • x-access-token: Token de Acceso a la entidad para la cual desea realizar la operación.
  • Content-Type: application/json

JSON Body ( POST )

Requeridos

  • total: Monto de la operación. Formato XXX.xx donde el PUNTO es el separador decimal.
  • currency: Moneda de la operación.
  • description: Descripción de la operación que será mostrada en el Checkout.
  • interval: Intervalo de Cobro de la suscripción.
  • trial: Cantidadd de Periodos de Prueba

Funcionamiento

  1. Crear una nueva suscripción.
  2. Guardar el ID de la Suscripción y el campo "shorten_url"
  3. Para crear un nuevo suscriptor redirigir al usuario a "shorten_url"

Respuesta

Tenga en cuenta que las respuestas del servicio siempre serán en formato JSON.

JSON

{
    "result": true,
    "data": {
        "_id": "5d0ce6df5b55192e471e94c3",
        "uid": "mv4vuUGYG",
        "type": "dynamic",
        "interval": "1m",
        "limit": 12,
        "trial": 0,
        "shorten_url": "http://mbbx.co/aQhnwh",
        "currency": "ARS",
        "status": "active",
        "description": "Mi suscripción 1",
        "total": 100,
        "created": "2019-06-21T14:17:03.157Z"
    }
}
