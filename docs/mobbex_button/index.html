Botón Mobbex

El botón Mobbex le permite embeber de manera fácil y rápida nuesta solución de checkout o Comercio Electrónico.

Creación

La inserción del botón Mobbex puede hacerse de 2 maneras, por un lado creando un checkout desde el lado Servidor o utilizando items previamente creados desde nuestra consola.

Creación de Checkout: esto debe hacerse desde el Servidor ya que sino quedarían expuestas las credenciales de su cuenta. Ver creación de Checkout

Creación de Items: esta técnica no requiere el uso de APIs, simplemente debe crear un item para vender desde su consola en la opción "Creá y Enviá botones o links de Pago" y luego copiar el ID del item generado.

Implementación

Insertar en el sitio

Cargá nuestra librería Javascript en tu sitio:

<html>
<body>
  <div id="mbbx-button"></div>
  <script type="text/javascript">
    var script = document.createElement('script');
    script.src = `https://res.mobbex.com/js/button/mobbex.0.9.14.js?t=${Date.now()}`;
    script.async = true;
    script.addEventListener('load', () => {
      renderMobbexButton();
    });
    document.body.appendChild(script);
  </script>
</body>
</html>

Renderizar el botón

Al renderizar podrás utilizar un checkout generado previamente en el servidor o un item. Tené en cuenta que el botón sólo aceptará uno de ellos.

function renderMobbexButton() {
  window.MobbexButton.render({
      checkout: `${server_generated_id}`,
      item: `${item_id}`,
      onPayment: (data) => {
        console.info('Payment: ', data);
      },
      onOpen: () => {
        console.info('Pago iniciado.');
      },
      onClose: (cancelled) => {
        console.info(`${cancelled ? 'Cancelado' : 'Cerrado'}`);
      },
      onError: (error) => {
        console.error("ERROR: ", error);
      }
  }, '#mbbx-button');
}

Creación y Apertura por Código

Al crear podrás utilizar un checkout generado previamente en el servidor o un item. Tené en cuenta que el botón sólo aceptará uno de ellos.

function renderMobbexButton() {
  var mbbxButton = window.MobbexButton.init({
      checkout: `${server_generated_id}`,
      item: `${item_id}`,
      onPayment: (data) => {
        console.info('Payment: ', data);
      },
      onOpen: () => {
        console.info('Pago iniciado.');
      },
      onClose: (cancelled) => {
        console.info(`${cancelled ? 'Cancelado' : 'Cerrado'}`);
      },
      onError: (error) => {
        console.error("ERROR: ", error);
      }
  });

  mbbxButton.open();
}

Opciones

Las siguientes opciones pueden ser provistas al botón Mobbex en la renderización o inicialización:

Inicializadores

checkout: ID del Checkout creado en el Servidor. No debe usarse en conjunto con item.
item: ID del Item generado en la consola Mobbex. No debe usarse en conjunto con Checkout.

Callbacks

onPayment: (data) => { }
onError: (error) => { }
onOpen: () => { }
onClose: (cancelled) => { }

Personalización

button: Objecto Javascript con las siguientes opciones

{
    text: 'Pagar con Mobbex',
    image: 'https://res.mobbex.com/images/sources/mobbex.png',
    backgroundColor: 'rgb(137,0,255)',
    textColor: 'rgb(255, 255, 255)',
    textSize: '1rem'
}
